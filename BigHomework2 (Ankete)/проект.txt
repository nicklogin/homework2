Интерактивная социолингвистическая анкета: проект

1. В файле variables.json лежат лингвистические переменные и их варианты.
2. В файле questions.xml лежат вопросы, для каждого из которых в тегах указывается
переменная из variables.json и контекст (напр. "formal").
3. В файле social_parameters.json лежат социальные переменные, для каждой из которых
указывается тип (количественная: количественная, качественная: дискретная
б недискретная).
4. Содержание файлов можно менять, главное, чтобы questions.xml не содержал вопросов на
линг. переменные, которых нет в variables.json
5. Можно использовать другие имена файлов, если передать их в качестве параметров объекту
класса Anketetext при его создании.
6. Для каждой диаграммы каждый раз генерируется новое имя, чтобы браузер не показывал
на странице '/stats' кэшированные диаграммы
